Las aplicaciones se encuentran distribuidas en sus carpetas respectivas, y se
pueden ejecutar con los siguientes comandos desde la raíz del proyecto (se
utilizó un sistema operativo macos):

CLIENTE:
python3 cliente/cliente.py

SERVIDOR INTERMEDIO:
python3 intermedio/servidor_intermedio.py

SERVIDOR ADIVINA:
go run adivina/main.go
